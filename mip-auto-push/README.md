# mip-auto-push

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-auto-push/mip-auto-push.js

```html

</mip-auto-push>
```

说明：{站点id}

# mip-qtkj-rem

mip-qtkj-rem 用来支持rem布局，移动端页面自适应

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-qtkj-rem/mip-qtkj-rem.js

## 示例

### 基本用法
```html
<mip-qtkj-rem class="dsb" width="750"></mip-qtkj-rem>
```

## 属性

### width

说明：rem布局对应的设计稿的宽度
必选项：是
类型：字符串
取值范围：无
单位：无
默认值：无

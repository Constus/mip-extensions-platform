# mip-zmall-countdown

倒计时组件

标题|内容
----|----
类型|业务组件
所需脚本|https://c.mipcdn.com/static/v1/mip-zmall-countdown/mip-zmall-countdown.js

## 示例

### 基本用法
```html
<mip-zmall-countdown data-endtime="1524153600"></mip-zmall-countdown>
```

## 属性

### data-endtime

说明：结束时间的时间戳
必选项：是
类型：字符串
取值范围：大于当前时间
单位：{单位}
默认值：无

## 注意事项

注意填写结束时间戳

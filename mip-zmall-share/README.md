## 示例

### 基本用法
```html
<mip-zmall-share data-btn-id="shareBtn">
    <mip-fixed type="bottom">
        <div class="mip-share-container">
            <mip-share></mip-share>
        </div>
    </mip-fixed>
</mip-zmall-share>
```

## 属性

### data-btn-id

说明：绑定的分享按钮     
必选项：是         
类型：String        
默认值：""

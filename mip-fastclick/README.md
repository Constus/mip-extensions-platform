# mip-fastclick

mip-fastclick 组件说明

简单封装fastclick库，使子组件 attach fastclick。

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-fastclick/mip-fastclick.js

## 示例

### 基本用法
只需将待 attach 的子组件使用 <mip-fastclick> 嵌套即可。

```html
<mip-fastclick>
    自定义内容，可以嵌套其他组件
</mip-fastclick>
```

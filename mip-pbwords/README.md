# mip-pbwords

mip-pbwords 关键词屏蔽

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-pbwords/mip-pbwords.js

## 示例

### 基本用法
```html
<mip-pbwords titl="名称" class="pbword" at="391k"></mip-pbwords>
```

## 属性

### titl

说明：标题
必选项：是
类型：字符串

### at

说明：网站归属
必选项：是
类型：字符串
默认值："391k"

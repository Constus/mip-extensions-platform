# mip-header-qiehuan

mip-header-qiehuan 点击一个元素,隐藏此元素,显示另一个元素,实现显示隐藏切换

标题|内容
----|----
类型|通用
支持布局|fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-header-qiehuan/mip-header-qiehuan.js

## 示例

### 基本用法
```
<mip-header-qiehuan>
	<div class="show"></div>
	<div class="hide"></div>
</mip-header-qiehuan>
```

# mip-anchor

mip-anchor 组件说明

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-anchor/mip-anchor.js

## 示例

### 基本用法
```html
<mip-anchor data-to="1">
    内容
</mip-anchor>
<p id='1'>
	内容
</p>
```

## 属性

### {属性名}

说明：data-to跟id是对应的，必须要填写的
必选项：是
类型：string
取值范围：a-zA-Z0-9
单位：无
默认值：null

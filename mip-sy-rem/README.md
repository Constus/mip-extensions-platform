# mip-sy-rem

mip-sy-rem 组件说明

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-sy-rem/mip-sy-rem.js

## 示例

### 基本用法
```html
<mip-sy-rem>
    自定义内容，可以嵌套其他组件
</mip-sy-rem>
```

## 属性

### {base}

说明：{设置字体大小的比例数，比例越大1rem表示的字体越小}
必选项：{否}
类型：{数字}
取值范围：{正整数}
单位：{无}
默认值：{32}

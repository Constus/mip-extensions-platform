# mip-laitui-ad

来推广告联盟广告JS组件。

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-laitui/mip-laitui-ad.js

## 示例

### 基本用法
```html
<mip-laitui-ad adz_id="123456">
    自定义内容，可以嵌套其他组件
</mip-laitui-ad>
```

## 属性

### adz_id

说明：广告ID
必选项：是
类型：数字
默认值：无

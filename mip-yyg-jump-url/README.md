# mip-yyg-jump-url

处理UC浏览器a标签跳转异常bug，代替A标签跳转功能

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-yyg-jump-url/mip-yyg-jump-url.js

## 示例

### 基本用法
```html
<mip-yyg-jump-url jumpUrl="tel:13092677466" >
    跳转按钮
</mip-yyg-jump-url>
```

## 属性

### {jumpUrl}

说明：{跳转地址}
必选项：{是}
类型：{string}

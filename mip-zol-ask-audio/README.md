
## 属性

### data-data
说明：音频数据（如果有此属性，则无需下面两条属性）
必选项：否
类型：json对象

### data-askid
说明：问题id
必选项：否
类型：数字

### data-replyid
说明：回答id
必选项：否
类型：数字

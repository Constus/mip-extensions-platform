# mip-btbcc-bdlxb

mip-btbcc-bdlxb 是一个调用百度离线宝功能的组件

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-btbcc-bdlxb/mip-btbcc-bdlxb.js

## 示例

### 基本用法
```html
<mip-btbcc-bdlxb sid="12088254"></mip-btbcc-bdlxb>
```

## 属性

### sid

说明：百度离线宝的代码sid
必选项：是
类型：字符串

## 注意事项

sid 不能为空。

# mip-gm-ad

mip-gm-ad 统计更美问答页面banner点击量

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|http://mipcache.bdstatic.com/static/mip-gm-ad/v1/mip-gm-ad.js

## 示例

```html
<mip-gm-ad href="">

</mip-gm-ad>
```

## 属性

### url

说明：领券接口地址
必选项：是
类型：string

### map-url

说明：查看地图链接
必选项：否
类型：string

### shopId

说明：店铺ID
必选项：是
类型：string

### coupons

说明：优惠券信息
必选项：是
类型：array

# mip-sy-news

mip-sy-news 组件说明

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-sy-news/mip-sy-news.js

## 示例

### 基本用法
```html
<mip-sy-news>

</mip-sy-news>
```

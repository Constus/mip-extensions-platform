# mip-xilu-ad

mip-xilu-ad 用来支持淘宝等第三方广告

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-xilu-ad/mip-xilu-ad.js

## 示例

### 基本用法
```html
<mip-xilu-ad type="taobao" token="5396689749"></mip-xilu-ad>
```

## 属性

### type

说明：广告类型  
必选项：是  
类型：字符串  
格式：必须为 taobao


### token

说明：广告ID  
必选项：是  
类型：字符串
